DstParser Documentation

Introduction

DstParser is a tool designed to parse and process DST (Data Summary Tape) files, commonly used in cosmic ray experiments. This documentation provides an overview of the installation, usage, and API of DstParser.

Note

This project is under active development. Contributions are welcome!

Installation

To install DstParser, use the following command:

pip install dstparser

Usage

Here is an example of how to use DstParser:

from dstparser import DstParser

parser = DstParser("path/to/dstfile.dst")
data = parser.parse()
print(data)
